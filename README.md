# Bulletin Board Client

## Requirements
- Node 14 or higher
- MySQL 5.7 or 8

## Installation

Clone the repo locally:
```
https://github.com/Myat-Theingi-Aung/bulletin-board-client.git
```

`cd` into cloned directory and install dependencies. run below command one by one.
```bash
npm install
```

## Server Run

Run the dev server:
```
npm run dev
```

Visit below url:
```
http://localhost:3000
```
